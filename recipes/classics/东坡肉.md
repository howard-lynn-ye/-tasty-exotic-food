# 东坡肉

> 以北宋文学家苏东坡命名，相传苏轼被贬杭州时所创。选用五花肉，经焯水、煎制、慢炖而成，色如玛瑙，肥而不腻，入口即化。

## 食材

| 食材 | 用量 |
|------|------|
| 五花肉 | 1000g |
| 绍兴黄酒 | 250ml |
| 酱油 | 4汤匙 |
| 老抽 | 1汤匙 |
| 冰糖 | 50g |
| 姜 | 4片 |
| 葱 | 4根 |

## 做法

1. 五花肉整块冷水下锅，焯水10分钟，捞出切成约5cm见方的块。
2. 砂锅底铺上葱段和姜片，将肉块皮朝下整齐摆入。
3. 加入黄酒、酱油、老抽、冰糖，液面没过肉的三分之二。
4. 大火烧开后，转最小火，加盖慢炖2小时。
5. 将肉块翻面（皮朝上），继续小火炖1小时。
6. 取出肉块装入小碗，浇上汤汁，上蒸锅蒸30分钟即可。

## 要点

- 选三层分明的五花肉，肥瘦比约6:4
- 黄酒要用绍兴黄酒，是灵魂所在
- 全程小火慢炖，"慢着火，少着水，火候足时它自美"
- 最后蒸制是关键一步，使肉质更加酥烂

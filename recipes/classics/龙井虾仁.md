# 龙井虾仁

> 杭州名菜，将西湖龙井茶与鲜河虾仁同烹，茶香清雅，虾仁玉白鲜嫩，色泽素雅，口感清爽。

## 食材

| 食材 | 用量 |
|------|------|
| 鲜河虾仁 | 250g |
| 龙井茶叶 | 5g |
| 鸡蛋清 | 1个 |
| 料酒 | 1汤匙 |
| 盐 | 适量 |
| 淀粉 | 1茶匙 |
| 植物油 | 适量 |

## 做法

1. 虾仁洗净沥干，加盐、蛋清、淀粉抓匀，腌制15分钟。
2. 龙井茶叶用80°C热水泡开，茶叶和茶汤分别留用。
3. 锅中倒油，烧至四成热（约120°C），放入虾仁滑散至变色，迅速捞出沥油。
4. 锅留少许底油，倒入虾仁，加料酒、少许茶汤翻炒。
5. 放入泡开的茶叶，快速翻炒均匀，出锅装盘。

## 要点

- 必须用明前龙井，茶香最佳
- 虾仁要鲜活河虾手剥，口感完全不同
- 油温不可过高，虾仁要滑嫩
- 全程快炒，动作要快，保持虾仁脆嫩

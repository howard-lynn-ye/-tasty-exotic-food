# 西湖醋鱼

> 杭州十大名菜之首，相传始于南宋，以西湖草鱼为主料，活杀现烹，浇以糖醋芡汁，色泽红亮，酸甜适口。

## 食材

| 食材 | 用量 |
|------|------|
| 草鱼 | 1条（约750g） |
| 姜 | 30g |
| 葱 | 2根 |
| 绍兴料酒 | 2汤匙 |
| 白糖 | 3汤匙 |
| 香醋 | 4汤匙 |
| 酱油 | 1汤匙 |
| 淀粉 | 1汤匙 |

## 做法

1. 草鱼去鳞去内脏，从背部剖开成两片，在鱼身上斜切数刀（不切断）。
2. 锅中加水烧开，加入姜片、葱段、料酒，将鱼片放入，大火煮3分钟后转小火，煮至鱼眼凸出、鱼肉熟透，捞出装盘。
3. 取少许鱼汤，加入白糖、香醋、酱油调匀，烧开后用水淀粉勾芡。
4. 将糖醋芡汁均匀浇在鱼身上，撒上姜末即可。

## 要点

- 鱼必须鲜活，现杀现烹
- 煮鱼火候很关键，过老则肉柴，过嫩则不熟
- 芡汁要亮而不厚，酸甜比例约4:3

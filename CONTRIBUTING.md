# 贡献指南 | Contributing Guide

> 欢迎所有留学生贡献你家乡的菜谱！不限菜系、不限国家，只要能在美国超市买到食材就行。
>
> *All international students are welcome to contribute recipes from your hometown! Any cuisine, any country — as long as ingredients are available at American supermarkets.*

---

## 如何贡献 | How to Contribute

### 1. Fork 本仓库 | Fork this repo
点击右上角 Fork 按钮，复制到你的 GitHub 账号下。

Click the Fork button in the top-right corner to copy this repo to your account.

### 2. 创建菜谱文件 | Create a recipe file
在 `recipes/` 对应分类目录下创建 `.md` 文件，按照下方模板编写。

Create a `.md` file in the appropriate `recipes/` subdirectory, following the template below.

### 3. 提交 Pull Request | Submit a PR
写好后提交 PR，我们会尽快 review！

Once done, submit a PR and we'll review it promptly!

## 提交要求 | Requirements

| 要求 | Requirement |
|------|-------------|
| 中英双语 | Must include both Chinese and English |
| 元数据 | Must include ⏱ time, 💰 cost, 🏷️ tags |
| 美国超市替代方案 | Must include American substitution table |
| 图片 (可选) | Image optional — Wikimedia Commons (CC BY-SA 4.0) preferred |
| 自己做过 | You should have actually cooked this dish |

# 杭州菜谱 | Hangzhou Recipes

杭州菜是浙江菜系的重要组成部分，以清鲜、脆嫩、爽滑著称，注重原汁原味，讲究时令食材。

## 经典名菜

- 西湖醋鱼
- 东坡肉
- 龙井虾仁
- 叫化童鸡
- 宋嫂鱼羹
- 西湖莼菜汤
- 杭州酱鸭
- 干炸响铃
- 片儿川
- 葱包桧

## 项目说明

本项目收集整理杭州传统菜谱，包括：

- **菜谱详情**：食材用量、烹饪步骤、技巧要点
- **文化背景**：每道菜的历史典故与文化渊源
- **食材指南**：时令食材选购建议
